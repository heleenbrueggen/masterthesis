Simple intraclass correlation

In multilevel data, the intraclass correlation is defined as \[\frac{\text{Var}(\beta_{0j})}{\text{Var}(Y_{ij})},\] where \(\text{Var}(\beta_{0j})\) is defined as the intercept variance (i.e., the variation in the group-specific means), and \(\text{Var}(Y_{ij})\) is defined as the total variation in the outcome variable \(Y_{ij}\) for subject \(i\) in group \(j\). Consider a model with a single level-one predictor \(X_{ij}\) and a single level-two predictor \(Z_{j}\). Consequently, we can define the following multilevel model, \[Y_{ij} = \beta_{0j} + \beta_{1j} X_{ij} + \varepsilon_{ij}.\] Here, \(\beta_{0,j}\) defines the random intercept, and can be decomposed into the following terms \[\beta_{0j} = \gamma_{00} + \gamma_{01} Z_j + \upsilon_{0j},\] with \(\gamma_{00}\) the average value of the intercept conditional on \(Z_j=0\), \(\gamma_{01}Z_j\) denotes the change in intercept dependent on \(Z_j\), and \(\upsilon_{0j}\) is a group-specific random error term that denotes each groups deviation from the expected intercept. The term \(\beta_{1j}\) can be decomposed in a similar manner \[\beta_{1j} = \gamma_{10} + \gamma_{11}Z_j + \upsilon_{1j},\] where all terms have a similar meaning, but then corresponding the slope, rather than the intercept.

Using this information, we can decompose the variance of \(Y_{ij}\) into the following terms \[ \text{Var}(Y_{ij}) = \text{Var}(\beta_{0j}) + \text{Var}(\beta_{1j}X_{ij}) + \text{Var}(\varepsilon_{ij}) + \text{Cov}(\beta_{0j}, \beta_{1j}X_{ij}) + \text{Cov}(\beta_{0j}, \varepsilon_{ij}) + \text{Cov}(\beta_{1j}X_{ij}, \varepsilon_{ij}). \]

If we generate \(X_{ij}\) and \(Z_{ij}\) independently of each other, using the following model \[ \begin{aligned} Z_{j} &\sim \mathcal{N}(\mu = 0, \sigma = 1) \\ X_{ij} & \sim \mathcal{N}(\mu = 0, \sigma = 2.5) \\ \gamma_{00} &= 1 \\ \gamma_{01} &= 0.5 \\ \upsilon_{0j} &\sim \mathcal{N}(\mu = 0, \sigma = 1.5) \\ \gamma_{10} &= 2 \\ \gamma_{11} &= 0.3 \\ \upsilon_{1j} &\sim \mathcal{N}(\mu = 0, \sigma = 0.7) \\ \varepsilon & \sim \mathcal{N}(\mu = 0, \sigma = 3) \end{aligned}, \] the variance \(\text{Var}(Y_{ij})\) is defined as \[ \begin{aligned} \text{Var}(Y_{ij}) &= \text{Var}(\beta_{0j}) + \text{Var}(\beta_{1j}X_{ij}) + \text{Var}(\varepsilon_{ij}) + \text{Cov}(\beta_{0j}, \beta_{1j}X_{ij}) + \text{Cov}(\beta_{0j}, \varepsilon_{ij}) + \text{Cov}(\beta_{1j}X_{ij}, \varepsilon_{ij}) \\ &= \{0.5^2 \cdot \text{Var}(Z_j) + \text{Var}(\upsilon_{0j})\} + \{[2^2 + 0.3^2 \cdot \text{Var}(Z_j) + \text{Var}(\upsilon_{1j})] \cdot \text{Var}(X_{ij})\} + \{\text{Var}(\varepsilon_{ij})\} \\ &= \{0.25 + 2.25\} + \{[4 + 0.09 + 0.49] \cdot6.25\} + 9 \\ &= 2.5 + 28.625 + 9 \\ &= 40.125. \end{aligned} \] Note that the covariance drop out of the equation because \(X_{ij}\) is generated independently of \(Z_j\) and the level two error terms \(\upsilon_{0j}\) and \(\upsilon_{1j}\), and because the residual errors are generated independently of everything. Now, letâ€™s check whether our formulation is actually correct. To do so, we specify a large number of groups \(J\), and a large number of observations within groups \(n_j\), so that our total estimate of the variance should converge towards the true value.

set.seed(9)

J  <- 5000
nj <- 500

g00 <- 1
g01 <- 0.5
ID  <- factor(rep(1:J, each = nj))
u0j <- rnorm(J, 0, 1.5) |> rep(each = nj)
Zj  <- rnorm(J, 0, 1) |> rep(each = nj)
b0j <- g00 + g01 * Zj + u0j
g10 <- 2
g11 <- 0.3
u1j  <- rnorm(J, 0, 0.7) |> rep(each = nj)
b1j <- g10 + g11 * Zj + u1j
Xij <- rnorm(J*nj, 0, 2.5)
eij <- rnorm(J*nj, 0, 3)

Yij <- b0j + b1j * Xij + eij
var(Yij)
[1] 40.20623

This is rather close to the true, specified variance of \(Y_{ij}\). Moreover, in the current set-up, the intraclass correlation can be calculated as \[ \frac{\text{Var}(\beta_{0j})}{\text{Var}(Y_{ij})} = \frac{2.5}{40.125} \approx 0.06 \]

Accordingly, the intraclass correlation can be specified by specifying the variance \(\text{Var}(\beta_{0j})\). Letâ€™s check whether we get similar results.

var(b0j) / var(Yij)
[1] 0.06044275
misty::multilevel.icc(Yij, cluster = ID)
[1] 0.06031085

The R-package misty reports a slightly different ICC, and Iâ€™m not entirely sure where this is due to (but the values are very close, and perhaps this is due to maximum likelihood estimation of the variances (but I donâ€™t know this for sure)).

Given that we have set the variance of the level two variable to \(\text{Var}(Z_j) = 1\), we can adjust the intraclass-correlation by either changing the variance of \(\upsilon_{0j}\), or by changing the slope \(\gamma_{01}\). Perhaps the best way to achieve this is by changing both variables with the same order of magnitude. If we set \(\gamma_{01} = 1.5\) and \(\Var{\upsilon_{0j} = 4.5^2\), the ICC can be calculated as \((1.5^2 + 4.5^2 = 22.5) / (22.5 + 28.625 + 9) \approx .37\). Letâ€™s change the variables g01 and u0j and generate \(Y_{ij}\) to check whether our results are indeed correct.

g01 <- 1.5
u0j <- rnorm(J, 0, 4.5) |> rep(each = nj)
b0j <- g00 + g01 * Zj + u0j
Yij <- b0j + b1j * Xij + eij
var(b0j) / var(Yij)
[1] 0.3744248

Of course, we could have achieved the same result by decreasing the variance of \(\beta_{1j}X_{ij}\) and/or the residual variance.

As a final check, letâ€™s see whether the R-package lme4 gives similar results.

fit <- lme4::lmer(Yij ~ 1 + (1 | ID), REML = FALSE)
sumfit <- summary(fit)
s_eij <- sumfit$sigma^2

s_b0j / (s_b0j + s_eij)
[1] 0.3740457

Apparently our results make sense.
